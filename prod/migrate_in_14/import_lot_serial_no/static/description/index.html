How to use

Import Lot/Serial Number in Incoming Shipment(Purchases).

Configuration

Go to Product and select Tracking 'By Lots' or 'By Unique Serial Number' Here we have selected 'By Lots'.


Now suppose you have made order of 21 quantity and its generated shipment for that purchase as shown below in fig.
Now if you want to Import it then you have to select the button which is highlighted in the below fig.


Once you click on the button, you will see one wizard which is shown below in the fig. Here you have to click on Import Lots/Serial Number Button.


Once you click on Import Lot/Serial Number button which we discussed above, you will get another wizard which is shown below in the fig.And you will get option to choose import serial number or lot number based on your product's tracking configuration.
When you choose import lot number then you have to import lot number with qty(Inside excel file), when you choose serial number then you have to import only serial number code (inside excel file , example is attached).


Once you click on Import in the above fig. that we discussed, you will get the following view which is shown below in the fig, this shows all set and all lot/serial numbers are imported perfectly.Now you have to just save it.


After clicking on save button you will get this view which is shown below in the fig. Here you have to click on Validate button.


Now we are selecting 'By Unique Serial Number' Tracking(if you have unique serial number for all of your product) which shown below in the fig.


Following the same process which we discussed for 'By Lots' for 'By Unique Serial Number'.
Only change is you have to select Serial Number option in wizard which is shown below in the fig.
