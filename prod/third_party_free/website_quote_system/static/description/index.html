Website Quote System

Website Quote System helps customer to create a quotation for any product on the website and can order the products in bulk with the custom prices and quantities.

Basic Features

  • Website Quote System helps customer to request a quote for any product from website.
  • Customers can order products in bulk with the custom price.
  • Admin can approve or reject the customer quotation.
  • Approved quotations can be easily added to cart by customer.
  • Admin can enable the quote system on their products.
  • Admin can set the minimum quantity and valid number of days for the products which are allowed to quote.
  • Customer can create a quotation with the custom price and quantity.
  • The customer can add description along with the price and quantity for products.
  • Approved quotations consist of an expiry date within which product is need to be buy.
  • Status of quotation changes on each action taken.
  • Admin can set the notification/email setting for each new quote created.
  • Facility of email notification to customer on change in status of its quotation.
