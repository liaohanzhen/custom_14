Marketplace Multi Step Vendor Signup

The module allows you to reconfigure the vendor registration form on the Odoo Marketplace. YOU can now redesign the entire vendor form into multiple steps to keep it clean and presentable. Each can have a different field to collect the relevant information from registering vendors.

Redesign your vendor registration form as you want

The module cleans up your vendor signup form on the Odoo marketplace.

Instead of adding all the fields on the same page, multi-step option allows you to categorize information into different steps.

So now you can design a neatly arranged multi-stepped registration form for vendors.

Collect Relevant information details from Vendors

Since the registration form fields can be separated into multiple steps, you can add much more fields without adding scrolls on the same page.

Add various fields at different steps to collect desired information from vendors registering for your Odoo marketplace.

Collect different specific types of information on each step. Such as;

Step/ Group Attributes / Fields
Personal info Name, address, etc.
Company info name of business, type of business, etc.
Contact info mail, mobile number phone number etc.
Seller Profile info profile url, offered sale commission, display name, etc.

Guide your vendors during signup on every step

Each step has a heading so you can define multiple steps with relevant heading to guide the vendor towards next steps while filling the form.

Features

  • The module allows you to redesign the vendor signup form on Odoo Marketplace.
  • You can divide up the original signup form into a multi-step registration form.
  • You can add new fields to the form from the backend.
  • Each field is created as an attribute in the Odoo backend.
  • The field can be set to as mandatory or optional in the form.
  • You can also add placeholder text to each field to assist the vendor during form filling.
  • The form can be divided into multiple steps containing their own fields for vendors to fill in the information.
  • Each step is managed as a Registration group which can have multiple fields in it.
  • Each step has a heading to neatly categorize the form on the Odoo marketplace.
  • You can set the sequence of each step from the Odoo backend.
