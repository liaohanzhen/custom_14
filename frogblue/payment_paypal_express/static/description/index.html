Website Paypal Express Checkout Payment Acquirer

PayPal Express Checkout with Smart Payment Buttons gives your buyers a simplified and secure checkout experience. PayPal intelligently presents the most relevant payment types to your shoppers, automatically, making it easier for them to complete their purchase using methods like Pay with Venmo, PayPal Credit, Credit Card payments, and other payment types.

Paypal Credentials

As this module provide Paypal Payment API integration thus first of all you will have to create an account on Paypal.

Create an app to receive REST API credentials for testing and live transactions.

Finally paste the client ID in Paypal Express Checkout Payment Acquire in Odoo .

Note: On production site switch your test mode to production and change the client ID too.
